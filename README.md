```bash

```

Fix macOS Mojave fonts

```bash
defaults write -g CGFontRenderingFontSmoothingDisabled -bool NO
```

Show hidden dotfiles in finder

```bash
defaults write com.apple.finder AppleShowAllFiles YES
killall Finder
```
